# 融易蓝（RYX Blue）设计规范

基于品牌色 `#2768FA` 的 H5 全局样式体系，定义于 `packages/ui/src/styles/globals.css`。

---

## 颜色 Token

| 代名词 | CSS 变量 | 色值 | Tailwind 类名 |
|--------|---------|------|--------------|
| 品牌蓝 | `--brand-primary` | `#2768FA` | `text-brand-primary` `border-brand-primary` |
| 标题黑 | `--brand-title` | `#010101` | `text-brand-title` |
| 按钮渐变起 | `--brand-btn-start` | `#33A1F9` | `from-brand-btn-start` |
| 按钮渐变止 | `--brand-btn-end` | `#2768FA` | `to-brand-btn-end` |
| 页头渐变起 | `--brand-header-start` | `#5099fe` | `from-brand-header-start` |
| 页头渐变止 | `--brand-header-end` | `#6aabff` | `to-brand-header-end` |
| 表单渐变起 | `--brand-form-header-start` | `#7AB1FF` | — |
| 表单渐变止 | `--brand-form-header-end` | `#F5F6F9` | — |

---

## 5 大套用模式

### 1. 融易蓝按钮（主操作按钮）

```
className="bg-gradient-to-r from-brand-btn-start to-brand-btn-end text-white"
```

从左到右 `#33A1F9 → #2768FA` 渐变蓝底白色文字。

### 2. 融易蓝页头（顶部导航栏）

```
className="bg-gradient-to-b from-brand-header-start to-brand-header-end"
```

从上到下 `#5099fe → #6aabff` 渐变。

### 3. 融易蓝标题

```
className="text-brand-title"
```

深黑色 `#010101`，用于浅色背景上的页面标题。

### 4. 融易蓝表单页（全屏表单背景）

```tsx
style={{ background: "var(--brand-form-header-gradient)" }}
```

顶部 0→200px 从 `#7AB1FF` 渐变到 `#F5F6F9`，200px 以下保持 `#F5F6F9`。

### 5. 融易蓝标签/链接

```
className="text-brand-primary"
className="border-brand-primary"
```

品牌蓝 `#2768FA`，用于选中态、标签文字、链接、对焦环。
