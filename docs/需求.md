# 需求

我需要搭建一个应用，这个应用包含三端：移动端h5、pad端(pad端尺寸在文档下方)、PC端的网页应用。请帮我考虑下pad端和PC端是否可以共用一套代码？

**pad尺寸**：
- 2025款 MatePad Pro 13.2英寸
  - 分辨率：2880 × 1920（3:2）
  - PPI：262；刷新率：144Hz；柔性OLED
-  2025款 MatePad Mini 8.8英寸
   - 分辨率：2560 × 1600（16:10，2.5K）
   - PPI：343；刷新率：120Hz；柔性OLED

**web构建**：
pnpm workspace monorepo

```
# pnpm-workspace.yaml
packages:
  - "apps/*"
  - "packages/*"
```

Directory layout:

```
<repo-root>/
├── apps/
│   ├── web/                        # @<scope>/web
│   │   ├── src/
│   │   │   ├── app/                # layout, pages, sitemap.ts, robots.ts
│   │   │   ├── components/{ui,<feature>}/
│   │   │   ├── content/            # MDX / JSON page content (multi-site reuse)
│   │   │   ├── config/{site,theme}.ts
│   │   │   └── lib/{env,api}.ts    # api.ts only when a backend is used
│   │   ├── package.json
│   │   └── .env.example
├── packages/
│   ├── <domain>/                   # @<scope>/<domain> — pure TS + tests (T1 tool sites)
│   └── shared-types/               # @<scope>/shared-types — DTO (T2/T3 ONLY)
├── scripts/                        # deploy / kill-port helpers (optional)
├── docs/
├── pnpm-workspace.yaml
├── package.json                    # root scripts orchestrate filters
├── .gitignore                      # committed; copy from gitignore.example on scaffold
├── CLAUDE.md                       # agent instructions for this repo
└── README.md
```
