# 目录结构

## 项目根目录

```
beeantmobile-main/projects/ryx/
├── src/                          应用源码
├── config_com.ronglvonline.app/  Capacitor / Android / iOS 打包配置
├── project.json                  Nx 构建 / serve / test 配置
├── tsconfig*.json                TypeScript 配置
├── jest.config.js                单元测试
├── publish-to-server-list.json   H5 发布服务器
└── *.bat                         发布脚本（H5 / APK）
```

## `src/` 结构

```
src/
├── main.ts                 入口：CoreHelper 初始化、bootstrap AppModule
├── config.ts               应用全局配置（API 域名、推送、微信等）
├── environments/           开发 / 生产环境开关
│   ├── environment.ts
│   └── environment.prod.ts
├── theme/                  Ionic 主题变量
├── assets/                 静态资源、i18n、iconfont、第三方 JS
└── app/                    业务模块（见下表）
```

## `src/app/` 主要模块

| 目录            | 职责                                                              |
| --------------- | ----------------------------------------------------------------- |
| **`tmc/`**      | **核心业务**：差旅预订（机票 / 酒店 / 火车 / 用车 / 订单 / 审批） |
| `tabs/`         | 底部 Tab 导航壳（首页 / 行程 / 我的）                             |
| `login/`        | 登录页                                                            |
| `account/`      | 账号管理                                                          |
| `member/`       | 成员、乘客、证件                                                  |
| `order/`        | 订单列表 / 详情 / 确认                                            |
| `registration/` | 注册                                                              |
| `workflow/`     | 审批流                                                            |
| `bpm/`          | BPM 流程                                                          |
| `hr/`           | HR（邀请、组织）                                                  |
| `mms/`          | 商城 / 供应链                                                     |
| `bank-card/`    | 银行卡                                                            |
| `password/`     | 密码修改                                                          |
| `message/`      | 消息                                                              |
| `public/`       | 公共页面                                                          |
| `services/`     | 应用级服务（login、storage、analyzer）                            |
| `components/`   | 跨模块公共组件                                                    |
| `directives/`   | 自定义指令                                                        |
| `guards/`       | 路由守卫                                                          |

## `tmc/` 子模块

| 子目录                     | 职责                                          |
| -------------------------- | --------------------------------------------- |
| `tmc-flight/`              | 国内机票搜索、列表、预订                      |
| `tmc-flight-dynamic/`      | 航班动态                                      |
| `tmc-hotel/`               | 国内酒店                                      |
| `tmc-international-hotel/` | 国际酒店                                      |
| `tmc-train/`               | 火车票                                        |
| `tmc-car/`                 | 用车 / 租车                                   |
| `tmc-rental-car/`          | 租车 POI 搜索                                 |
| `tmc-order/`               | TMC 订单                                      |
| `tmc-confirm-information/` | 下单确认                                      |
| `tmc-checkout-success/`    | 支付成功                                      |
| `tmc-approval-task/`       | 审批任务                                      |
| `tmc-switch-company/`      | 切换 TMC 公司                                 |
| `tmc-select-passenger/`    | 选择乘客                                      |
| `components/`              | TMC 共享组件                                  |
| `tmc.service.ts`           | TMC 核心服务（机场/车站缓存、公司、申请单等） |
| `ryx-routing.module.ts`    | 融易行定制路由                                |

## `config_com.ronglvonline.app/`

| 文件 / 目录             | 职责                          |
| ----------------------- | ----------------------------- |
| `capacitor.config.json` | Capacitor AppId、原生工程路径 |
| `build.json`            | Android 签名 keystore         |
| `prod-config.ts`        | 生产环境 CONFIG 覆盖          |
| `ApiConfig.json`        | 部署环境 API 端点             |
| `publish-servers/`      | 测试 / 生产发布服务器列表     |
| `android/`、`ios/`      | 原生工程资源                  |
