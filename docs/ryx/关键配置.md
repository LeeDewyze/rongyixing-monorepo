# 关键配置文件

## 配置一览

| 文件                                                | 作用                                                                 |
| --------------------------------------------------- | -------------------------------------------------------------------- |
| `src/config.ts`                                     | **核心配置**：API 域名、App 名称、推送 Key、微信 AppId、隐私协议 URL |
| `src/environments/environment.ts`                   | 开发 / 生产开关（`production`、`mockProBuild`）                      |
| `config_com.ronglvonline.app/capacitor.config.json` | Capacitor：`appId`、原生工程路径、webDir                             |
| `config_com.ronglvonline.app/build.json`            | Android 签名 keystore                                                |
| `config_com.ronglvonline.app/prod-config.ts`        | 生产环境 `CONFIG` 覆盖                                               |
| `config_*/ApiConfig.json`                           | 各部署环境的 API 端点                                                |
| `config_*/publish-servers/*.json`                   | 测试 / 生产发布服务器列表                                            |
| `project.json`                                      | Nx 构建目标（outputPath、styles、scripts、fileReplacements）         |
| `tsconfig.app.json`                                 | TypeScript 编译配置                                                  |
| `publish-to-server-list.json`                       | H5 发布服务器清单                                                    |

---

## `src/config.ts`

应用运行时主配置，导出 `CONFIG: AppConfig`。

### 关键字段

| 字段                                          | 说明                                                    |
| --------------------------------------------- | ------------------------------------------------------- |
| `mockProBuild`                                | `true` 时 prod 构建仍走测试库地址                       |
| `production`                                  | 是否生产模式                                            |
| `appTitle`                                    | 应用名称（融易行）                                      |
| `appDomain.production`                        | 生产域名 `rongtrip.cn`                                  |
| `appDomain.debug`                             | 测试域名 `rtesp.cn`                                     |
| `getApiUrl()`                                 | 返回 `https://app.rongtrip.cn` 或 `http://app.rtesp.cn` |
| `wechat`                                      | 微信 AppId、Universal Links                             |
| `miPush` / `hwPush` / `oppoPush` / `vivoPush` | 厂商推送参数                                            |
| `apiTimemoutTime`                             | API 超时（默认 30s）                                    |
| `getPrivacyUrl()` / `getUserAgreementUrl()`   | 隐私政策与用户协议 URL                                  |

---

## `src/environments/environment.ts`

Angular 环境文件，构建时可通过 `fileReplacements` 替换为 `environment.prod.ts`。

| 字段              | 说明                |
| ----------------- | ------------------- |
| `production`      | 是否生产构建        |
| `localhost`       | 本地开发标记        |
| `disableNetWork`  | 禁用网络（调试）    |
| `enableLocalData` | 启用本地数据        |
| `mockProBuild`    | 与 `config.ts` 联动 |

---

## `capacitor.config.json`

```json
{
  "appId": "com.ronglvonline.app",
  "appName": "融易行",
  "webDir": "www_com.ronglvonline.app",
  "android": { "path": "android_com.ronglvonline.app" },
  "ios": { "path": "ios_com.ronglvonline.app" }
}
```

控制 Capacitor 原生壳与 Web 产物目录的对应关系。

---

## `project.json`

Nx 项目定义，主要配置：

- **build**：`@angular-devkit/build-angular:browser`，输出 `dist/apps/ryx`
- **serve**：开发服务器
- **production configuration**：启用 AOT、优化，替换 `environment.prod.ts`
- **assets / styles / scripts**：全局样式、iconfont、verifycode 等第三方脚本

---

## 环境切换

| 场景     | 方式                                                 |
| -------- | ---------------------------------------------------- |
| 开发 H5  | `ng serve` / Nx `serve` target                       |
| 测试 API | `CONFIG.mockProBuild = true` 或测试域名              |
| 生产 API | `CONFIG.production = true` 且 `mockProBuild = false` |
| 切换脚本 | 根目录 `切换生产环境.bat`                            |
| 发布 H5  | `生产发布-H5.bat`                                    |
| 发布 APK | `生产发布-APK.bat` / `发布测试-APK.bat`              |

---

## 与新 monorepo 环境变量对照

| Legacy                    | 新 monorepo（`apps/h5`）                 |
| ------------------------- | ---------------------------------------- |
| `CONFIG.getApiUrl()`      | `VITE_API_BASE_URL`                      |
| `CONFIG.appDomain`        | `VITE_API_DOMAIN`                        |
| `mockProBuild` / 模式切换 | `VITE_API_MODE`（mock / proxy / direct） |
| Capacitor `appId`         | `VITE_APP_ID`                            |

详见 `apps/h5/.env.example` 与 [h5-login-troubleshooting.md](../h5-login-troubleshooting.md)。
