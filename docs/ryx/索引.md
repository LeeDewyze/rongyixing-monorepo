## 一句话总结

基于 **Angular + Ionic + Capacitor** 的企业差旅混合 App，采用 **Nx Monorepo + 功能模块分层** 架构，核心业务是 **机票 / 酒店 / 火车 / 用车预订 + 差旅审批 + 订单支付**，通过 Beeant Proxy 与后端 TMC 服务通信。
